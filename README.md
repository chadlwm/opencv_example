# What you will earn

- opencv basic usage
- how to use g++ to build your cpp
- how to use cmake to build your cpp, including one cpp or multi cpp.

GOOD LUCK.


# How to build

support two build ways: `g++` or `cmake`

just run: 

```
./build_with_g++.sh
```

or 

```
./build_with_cmake.sh
```

you will get execute file in build dir.
